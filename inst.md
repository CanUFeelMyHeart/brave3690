# ***MarkDown***

## **Стилизация текста**

**Полужирный текст**
```sh
** текст **
```

*Курсив*
```sh
* текст *
```
Ссылка 
```sh
[...](ссылка)
```
[Basic Syntax](https://www.markdownguide.org/basic-syntax/)

>Блок текста
```sh
>Text
```
>В одном блоке 
>>можно сделать блок внутри 
```sh
>text \enter\ >>text
```

`текст поменьше`
```sh
`text`
```
URL & Email можно добавлять так:
```sh
<text>
```
<https://www.markdownguide.org/basic-syntax/>

Чтобы добавить картинку можно:
```sh
![название](имя картинки.jpeg)
```

Или вставить ссылку
```sh
![имя картинки](ссылка на картинку)
```

Нумерация или список

Нумерация:
1. Что-то тут
2. Что-то тут 
    - Что-то внутри 2 что-то
    - Что-то еще внутри 2 что-то
3. что-то

*Список:*

```sh
1. Name
2. Name
    - Name
    - Name
3. Name
```
