# Подсказка по Git

Создание репозитория:
```sh
git init
```
Просмотр файлов в репозитории
```sh
dir
```
Просмотр статус репозитория
```sh
git statusht
```

Добавить к индексации
```sh
git add
```
Зафиксировать изменения
```sh
git commit - m "Message text"
```
Посмотреть лог в полном виде
```sh
git log
```

Посмотреть лог в сокращенном виде
```sh
git log --oneline
```

Перемещение по веткам
```sh
git checkout <имя ветки>
```
Отображение всех веток
```sh
git branch
```
Создание новой ветки
```sh
git branch <имя ветки>
```
Удаление ветки
```sh
git git branch -d <имя ветки>
```
Слияние с *текущей* веткой 
```sh
git merge <имя ветки для слияния с текущей веткой>
```
Визуализация всех веток
```sh
git log --graph
```

копирование файла с github
```sh
git clone cсылка c github для выгрузки
```
Скачивание из текущего репозитория
```sh
git pull
```
Отправка своей версии репозитория во внешний репозиторий
```sh
git push
```
