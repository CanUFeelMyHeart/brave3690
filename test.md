# Подсказка по GIT

Команда создания репозитория:
```sh
git init
```
Команда отображения текущего статуса репозитория:
```sh
git status
```
Команда отслеживания изменений:
```sh
git diff
```
Команда добавления изменений в репозиторий:
```sh
git add
```
Команда создания коммита:
```sh
git commit -m "Message"
```
Команда выведения списка всех коммитов:
```sh
git log
```
Команда выведения списка всех коммитов сокращенно:
```sh
git log --oneline
```
Команда переключения по коммитам:
```sh
git checkout <commit_adress>
```
Перемещение по веткам:
```sh
git checkout branch_name
```
Отображение всех веток:
```sh
git branch
```
Создание новой ветки:
```sh
git branch branch_name
```
Комганда удаления ветки:
```sh
git branch -d branch_name
```
Выведение лога коммитов со слиянием:
```sh
git log --oneline --graph
```
Команда клонирования
```sh
git clone
```
Команда наполнения репозитория другим репозиторием
```sh
git pull
```
Команда передачи изменений из локального файла в удаленный
```sh
git push
```


# Подсказки по командной строке

Команды смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. MacOs, Linux
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файла в Windows:
```
del <filename>
```
в Linux, MacOs:
```sh
rm <filename>
```
