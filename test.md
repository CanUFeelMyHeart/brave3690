## *Команды GIT
Инициализация GIT (в нужном каталоге в режиме командной строки)
```sh
git init
```

Добавление файла инструкции в GIT
```sh
git add GB_Help.md
```

Коммит изменений
```sh
git commit -m "Создаем коммит"
```

Посмотреть статус GIT
```sh
git status
```

Просмотр лога GIT
```sh
git log
```
