# Подсказки по командной строке

*Комманды смены директории*
```sh
cd cL\folder_name
```

*Команда отображения текущей директории*
```sh
pwd
```

*Листинг текущей директории Windows:*
```sh
dir
```
![Пример](Resources/Images/Cmd_dir.JPG)

*Lunix, MacOs*
```sh
ls
```

*Удаление файла в Windows:*
```sh
del <filename>
```

*в Lunix, MacOs:*
```sh
rm <filename>
```
